# Design: Legal Pages

## Component: Privacy Policy
**File**: `app/privacidade/page.jsx`
- **Layout**: Centered, max-width container (e.g., `max-w-4xl`) for readable typography.
- **Theme**: `#02040a` background with white/gray text (`text-gray-300`).
- **Header**: "Política de Privacidade" with a `lucide-react` icon (e.g., `ShieldCheck`).
- **Content Sections**:
  1. Data Collection (Name, Course, Shift, Tasks).
  2. Data Storage (Supabase).
  3. AI Processing (Google Gemini API - explicit mention that data is used strictly for generating academic insights and not for training generic models).
- **Navigation**: A "Voltar para o Início" button (`ArrowLeft` icon) at the top or bottom linking to `/`.
