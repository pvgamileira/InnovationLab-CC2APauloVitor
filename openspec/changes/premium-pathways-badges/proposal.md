# Proposta: Atalhos Premium e UI de Insígnias de Conquista

## Problema
Queremos melhorar a visibilidade e o acesso à nossa página Premium (EduTrack Pro) tanto para novos visitantes quanto para os usuários já cadastrados no painel, além de enriquecer a gamificação do dashboard com conquistas visuais elegantes (Insígnias/Badges).

## Solução Proposta
1. **Atalhos Premium (Pathways):**
   - **No Dashboard (`app/dashboard/layout.jsx`):** Adicionar um botão flutuante e brilhante com sweep de luz contendo o rótulo *"👑 Upgrade Pro"* na base da lista de navegação.
   - **Na Landing Page (`app/page.jsx`):** Adicionar o botão *"Ver Planos"* ao lado do botão primário "Começar Agora" na seção Hero, rolando suavemente para a seção de preços.
2. **UI de Insígnias de Conquista:**
   - **No Dashboard (`app/dashboard/page.jsx`):** Ao lado do indicador de Nível e XP do estudante no cabeçalho do painel, adicionar a seção de "Insígnias" com 3 medalhas estilizadas em glassmorphism e tooltips interativos no hover (`Zap` para Foco Rápido, `Target` para Sniper de Prazos e `ShieldCheck` para Zero Atrasos).

## Benefícios
- **Conversão Otimizada:** Melhorias de navegação direta para a área comercial.
- **Engajamento e Retenção:** Conquistas simbólicas recompensadoras que incentivam o aluno a se manter ativo e organizado nas suas disciplinas.
