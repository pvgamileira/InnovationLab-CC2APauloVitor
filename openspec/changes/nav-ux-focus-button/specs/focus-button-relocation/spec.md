## ADDED Requirements

### Requirement: Top-Right Focus Button
The system SHALL display the "Modo Foco" button at the top-right header of the Disciplinas page.

#### Scenario: Navigating to focus mode
- **WHEN** the user clicks the "Modo Foco" button in the Disciplinas header
- **THEN** the user is redirected to the Foco page

## REMOVED Requirements

### Requirement: Subject Card Focus Button
**Reason**: Clutters the UI and is aesthetically displeasing inside the Subject cards.
**Migration**: Users will access Focus Mode globally via the Disciplinas page header, the sidebar, or mobile dashboard quick actions.
